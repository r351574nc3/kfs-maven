60      /**
61       * Gets the apo limit for the given parameters using the following logic:<br>
62       * <br>
63       * First it checks to see if an existing {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization} object exists for the
64       * associated parameters. If one exists and it is not excluded (see
65       * {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization#isVendorContractExcludeIndicator()}) this will return the value
66       * of {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization#getVendorContractPurchaseOrderLimitAmount()}.<br>
67       * <br>
68       * If an associated {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization} object cannot be found then a valid
69       * {@link org.kuali.kfs.vnd.businessobject.VendorContract} object will be sought. If one is found this method will return the value of
70       * {@link org.kuali.kfs.vnd.businessobject.VendorContract#getOrganizationAutomaticPurchaseOrderLimit()}.<br>
71       * <br>
72       * If no valid {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization} or
73       * {@link org.kuali.kfs.vnd.businessobject.VendorContract} objects can be found for the given parameters this method will return null.
74       * 
75       * @param contractId id used to find {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization} object and
76       *        {@link org.kuali.kfs.vnd.businessobject.VendorContract} object
77       * @param chart chart code for use in finding {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization} object
78       * @param org org code for use in finding {@link org.kuali.kfs.vnd.businessobject.VendorContractOrganization} object
79       * @return the automatic purchase order limit amount from the contract found using the parameters. If parameters do not find
80       *         valid vendor contract objects then null is returned.
81       */
82      public KualiDecimal getApoLimitFromContract(Integer contractId, String chart, String org);
