33      /**

38       * @param dataFile the data file. A connection to this file may be opened multiple times by this method.

42       * @param feederProcessName the name of the process that's invoking this method.
43       * @param reconciliationTableId the name of the reconciliation block to use within the reconciliation file
44       * @param statusAndErrors a class with references to a {@link org.kuali.kfs.gl.batch.service.impl.EnterpriseFeederStatus} object and a list

46       *        which means that it can't return a value. This parameter allows the method to output status/error information
47       */
